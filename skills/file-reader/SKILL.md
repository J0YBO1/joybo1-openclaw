# File Reader Skill

## 功能
这是一个文件读取技能包，支持读取多种文件格式并提供友好的预览界面。

## 支持的文件格式
- 📝 **文本文件**：.txt, .md, .json, .js, .html, .css, .xml
- 📊 **数据文件**：.csv, .tsv
- ⚙️ **配置文件**：.yaml, .yml, .toml, .ini
- 🔧 **代码文件**：.py, .java, .cpp, .go, .rs

## 使用方法

### 基本用法
```
file-reader --path "文件路径"
```

### 带参数调用
```
file-reader --path "data.json" --format json --lines 20
```

### 参数说明
- `--path`：要读取的文件路径（必需）
- `--format`：文件格式（可选，自动检测）
- `--lines`：预览行数（默认：50行）
- `--encoding`：文件编码（默认：utf-8）

## 工作流程
1. 接收用户输入的文件路径和参数
2. 验证文件是否存在和可读
3. 根据文件格式选择合适的读取方式
4. 处理文件内容（预览、格式化等）
5. 返回读取结果和统计信息
