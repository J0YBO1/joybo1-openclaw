# PDF Reader Skill

## 功能
这是一个PDF文件读取技能包，专门用于读取PDF文件内容，提取文本、元数据和页面信息。

## 支持的功能
- 📝 **文本提取**：从PDF中提取所有文本内容
- 📊 **元数据读取**：读取PDF的作者、标题、创建日期等信息
- 📄 **页面信息**：获取PDF页数、页面尺寸等信息
- 🔍 **内容预览**：提供PDF内容的预览和摘要

## 使用方法

### 基本用法
```
pdf-reader --path "文件路径.pdf"
```

### 带参数调用
```
pdf-reader --path "document.pdf" --pages 1-5 --extract-text
```

### 参数说明
- `--path`：要读取的PDF文件路径（必需）
- `--pages`：指定要读取的页面范围，如"1-5"或"1,3,5"（默认：所有页面）
- `--extract-text`：提取文本内容（默认：是）
- `--metadata`：显示元数据信息（默认：是）
- `--preview-lines`：预览文本行数（默认：100行）

## 工作流程
1. 接收用户输入的PDF文件路径和参数
2. 验证PDF文件是否存在和可读
3. 使用PDF解析库读取文件内容
4. 根据参数提取文本、元数据或页面信息
5. 格式化输出结果，提供友好的预览界面

## 开发说明
- 这是一个专门的PDF文件读取技能包
- 需要安装PDF解析库（如pdf-parse或pdf.js）
- 支持中英文PDF文档
- 处理加密或受保护的PDF文件时会有相应提示

## 技术要点
- PDF文件解析和文本提取
- 元数据读取和处理
- 大PDF文件的分页处理
- 错误处理和用户友好提示
- 支持多种PDF编码格式
