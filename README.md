## 🚀 快速开始

### 环境要求
- Node.js >= 16.0.0
- OpenClaw 运行环境
- Python 3.8+（部分技能需要）

### 使用示例
```bash
# 克隆本仓库
git clone https://github.com/J0YBO1/joybo1-openclaw.git

# 进入技能目录
cd joybo1-openclaw/skills/hello-world

# 测试技能
node test.js
```

### 技能包开发
每个技能包包含：
1. `SKILL.md` - 技能文档（OpenClaw规范）
2. `index.js` - 技能实现代码
3. `package.json` - 包配置
4. `test.js` - 测试文件
