# Hello World Skill

## 功能
这是一个简单的Hello World技能包，用于验证OpenClaw技能开发环境和工作流程。

## 使用方法

### 直接调用
```
hello-world
```

### 带参数调用
```
hello-world --name "黄冠"
```

## 工作流程
1. 接收用户输入（可选姓名参数）
2. 生成个性化的问候语
3. 返回问候结果
